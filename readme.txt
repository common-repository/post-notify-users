== Description ==

= Notified by e-mail to the user with the roles set at the time of new posting. =
* Can specify the roles of the user to be notified.
* Can specify the custom post to be notified.

== Installation ==

1. Upload `post-notify-users` directory to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
